Saudi Embassy Attestation

Documents required for Saudi Attestation on Educational Certificate

  • Original Degree Certificate(should be attested from Human Resource Department of the concerned state)
  • Original Offer Letter with Chamber And Foreign Ministry Attestation (need minimum 180 days validity )
  • Passport Copy ( Clear and clean both side ) ( black/color )
  • Iqama Copy (If already working in Saudi Arabia) ( Color/ black )
  • Visa Copy (if already working in Saudi Arabia)
  • Authorization Letter
  • All mark sheet copies
